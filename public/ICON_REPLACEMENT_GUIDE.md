## Quick Steps:

1. Save the logo image from the media reference to `public/it-logo-source.png`
2. Go to: https://realfavicongenerator.net/
3. Upload `public/it-logo-source.png`
4. Download the generated icons
5. Replace these files in `public/`:
   - `icon-192.png` (192x192)
   - `icon-512.png` (512x512)
   - `apple-touch-icon.png` (180x180)
   - `favicon.ico` (multi-size: 16x16, 32x32, 48x48)

## Alternative: Manual Resize

If you have image editing software:
- Maintain aspect ratio
- Center on transparent or black background if needed
- Export as PNG (except favicon.ico which should be ICO format)
